# 三、硬件配置分级要求

软件启动自动全维度硬件检测（耗时≤10秒），根据检测结果匹配对应配置等级，加载专属推理策略与功能权限，未达基础配置给出精准整改提示，核心前提：必须为SSD固态硬盘+Windows 10/11 64位；同时适配单张图片生成数字人、自定义音色推理的硬件需求。

## 3.1 基础能用配置（入门级，纯CPU/入门NVIDIA显卡，个人尝鲜/低频次使用）

- CPU：Intel i5 6代+/AMD Ryzen 5 1600+（≥4核8线程，主频≥2.5GHz）

- 内存：≥8GB RAM（DDR4 2400+，单/双通道均可）

- 显卡：无要求（纯CPU推理）/NVIDIA入门卡（GTX 1050 Ti/MX550，显存≥4GB，支持CUDA）

- 硬盘：SSD≥128GB，空闲空间≥15GB（安装软件+模型存储+缓存+作品）

- 显卡驱动：有NVIDIA显卡则≥470.xx版本，无显卡则无要求

- 功能限制：单次批量制作≤10条文案，自定义数字人/音色模型单文件≤50MB；单张图片生成数字人耗时≤60秒，禁止多任务并行

# 四、产品整体架构（含技术架构联动）

## 4.1 界面布局

采用「左侧固定功能菜单+右侧核心内容区」Windows桌面端经典布局，左侧为8大核心模块入口，右侧为对应模块的操作界面，整体风格简洁明了、无冗余，符合Windows用户操作习惯，支持浅色/深色主题一键切换，切换后实时生效；新增单张图片生成数字人入口、自定义音色上传入口，布局贴合原有操作逻辑，不增加用户学习成本。

## 4.2 核心模块划分（8大模块，独立联动，围绕视频制作核心场景）

1. 首页模块：软件唯一入口，提供核心功能快速跳转、最近操作记录、新手教程引导，实时展示硬件状态与GPU加速情况。

2. 单条制作模块：核心功能模块，采用5步步骤化引导+右侧实时预览布局，实现单条数字人口播视频全流程制作，集成单张图片生成数字人、自定义音色选择功能。

3. 批量制作模块：支持文案批量导入、统一配置参数（含自定义音色、数字人），本地批量生成多条口播视频，适配批量创作需求，按硬件等级限制单次生成条数。

4. 数字人管理模块：管理官方数字人、单张图片生成的数字人、用户自定义上传的数字人，支持预览、收藏、使用、删除、重新生成/适配，确保口型同步效果。

5. 音色管理模块：管理内置CosyVoice3-0.5B衍生音色与用户自定义上传音色，支持试听、收藏、使用、上传、适配、删除、重新加载，实现模型本地加载与管理，释放硬盘空间。

6. 作品库模块：本地统一管理所有生成的口播视频，支持查看、播放、编辑、删除、批量操作，重新编辑可精准加载原始配置（含数字人、音色参数）。

7. 设置模块：配置软件运行参数、修改模型/视频存储路径（全英文路径）、优化性能设置、查看硬件信息、清理缓存、重置所有设置。

8. 帮助与反馈模块：提供本地离线使用教程、常见问题（FAQ）解答、客服入口、问题反馈通道，含单张图片生成数字人、自定义音色上传的常见问题解决方案。

## 4.3 核心技术流程（本地全离线，无云端依赖，全流程本地完成）

1. 文案处理：用户输入/导入文案 → 本地文案优化引擎（自动断句、口语化优化、字数校验，耗时≤3秒）；

2. 数字人生成/选择：用户选择「官方数字人/单张图片生成数字人/自定义上传数字人」→ 对应技术引擎处理（图片转数字人/模型适配）→ 输出可驱动数字人模型；

3. 语音合成：用户选择音色（内置/自定义）与参数 → 内置CosyVoice3-0.5B模型或自定义音色模型本地推理（生成带情感的WAV语音文件，CPU/GPU自适应加载）；

4. 口型同步：本地Wav2Lip口型同步引擎（将语音与数字人视频匹配，口型同步误差≤0.1秒，NVIDIA GPU加速）；

5. 视频合成：用户配置背景、字幕、BGM等参数 → 本地FFmpeg视频合成引擎（整合所有素材，NVIDIA NVENC硬件编码，生成1080P MP4视频）；

6. 作品管理：生成的视频自动保存至本地指定英文路径，同步至作品库，支持后续编辑、播放、导出、删除等操作。

## 4.4 核心技术架构（分层设计，支撑功能落地）

整体技术架构分为5层，自上而下实现「交互入口→适配处理→核心引擎→模型支撑→存储管理」全链路本地化，联动所有功能模块，确保单张图片生成数字人、自定义音色上传等核心功能可落地，各层职责清晰、低耦合。

1. 交互适配层：承接所有模块交互需求，负责单张图片上传、自定义音色上传等操作的前端适配，以及不同硬件配置的动态适配，传递用户指令至核心引擎层，反馈处理结果至前端。

2. 核心引擎层：核心技术支撑，包含口型同步引擎（Wav2Lip轻量优化版）、数字人渲染引擎（Unity轻量版）、自定义数字人适配引擎、单张图片生成数字人引擎（SadTalker轻量化版）、自定义音色适配引擎、视频合成引擎（FFmpeg）。

3. 模型支撑层：内置核心模型，包含CosyVoice3-0.5B语音模型、Wav2Lip口型驱动模型、SadTalker图片转数字人模型、官方数字人基础模型、渲染优化模型，支持自定义模型本地加载与校验。

4. 格式解析层：负责数字人（FBX/GLB/OBJ/图片）、音色相关文件的格式解析与转换，将用户上传的文件转换为引擎可识别的专用格式，完成轻量化优化。

5. 存储管理层：负责所有模型、素材、作品的本地存储（全英文路径），支持模型缓存清理、冗余文件删除，采用索引机制提升加载速度，确保本地读取延迟≤1秒。

## 5.2 单条制作模块（核心功能模块，含核心技术落地）

### 5.2.1 功能描述

采用“左侧5步步骤化引导+右侧实时预览区”的布局，实现单条数字人口播视频的全流程制作，步骤间可自由回退，所有配置修改1秒内同步至预览区；集成单张图片生成数字人、自定义音色选择功能，技术上适配本地离线推理，兼顾易用性与性能。

### 5.2.2 界面布局

- 左侧：固定5步制作流程，依次为「文案输入→语音选择→数字人选择（含单张图片生成）→视频设置→生成视频」，当前操作步骤高亮显示，已完成步骤标注“√”，支持点击任意已完成步骤回退，回退后保留所有配置。

- 右侧：实时预览区，占界面70%以上空间，同步展示当前步骤的配置效果（含图片生成的数字人预览、自定义音色语音预览），支持语音播放、视频预览，所有参数修改1秒内更新预览效果。

### 5.2.3 分步功能详情（含技术解决方案）

#### 步骤1：文案输入（核心：文案处理、校验、优化）

1. 文案输入框：支持手动输入、粘贴文本，可自由拉伸，支持换行，最大输入字数1000字，实时显示当前字数；技术适配：输入文本实时缓存至本地，防止软件异常退出丢失内容，缓存文件存储至指定英文路径。

2. 实时字数提示：输入框下方显示当前字数，标注适配建议，字数异常时标红提示；技术适配：字数校验本地实时执行，无需调用云端接口，响应延迟≤0.1秒。

3. 文案快捷操作按钮：清空、复制、粘贴，点击后有对应提示，避免误操作；技术适配：剪贴板操作调用Windows本地接口，兼容所有Windows系统自带剪贴板功能，无兼容性问题。

4. 智能文案优化：输入框右侧“优化”按钮，点击后执行自动断句、口语化优化，耗时≤3秒，超过3秒显示加载动画；技术适配：采用本地轻量文案优化模型（体积≤10MB），内置常用口语化语料库，无云端依赖，优化后支持还原。

5. 口播文案模板库：输入框下方“模板”按钮，弹出5类可商用模板，点击自动加载至输入框，支持自定义修改；技术适配：模板本地存储至安装包目录（英文路径），加载延迟≤1秒，无需额外下载。

6. 下一步按钮：仅当文案字数≥10字时可点击，点击后跳转至步骤2，保留当前所有配置；技术适配：跳转时自动保存当前文案配置，与后续步骤配置关联，确保流程连贯。

#### 步骤2：语音选择（核心：基于CosyVoice3-0.5B模型，支持自定义音色上传与选择）

1. 音色分类展示：按“我的收藏、男声、女声、情感音、方言音、自定义音色”划分，每个音色卡片包含名称、描述，自定义音色分类单独标注；技术适配：音色列表采用本地索引加载，从指定英文路径读取音色模型信息，加载延迟≤1秒，支持批量加载。

2. 音色核心操作：试听、选择、收藏，试听延迟≤1秒，语音清晰无杂音；技术适配：
