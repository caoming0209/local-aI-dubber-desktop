# 三、硬件配置分级要求

软件启动自动全维度硬件检测（耗时≤10秒），根据检测结果匹配对应配置等级，加载专属推理策略与功能权限，未达基础配置给出精准整改提示，核心前提：必须为SSD固态硬盘+Windows 10/11 64位。

- CPU：Intel i5 6代+/AMD Ryzen 5 1600+（≥4核8线程，主频≥2.5GHz）

- 内存：≥8GB RAM（DDR4 2400+，单/双通道均可）

- 显卡：无要求（纯CPU推理）/NVIDIA入门卡（GTX 1050 Ti/MX550，显存≥4GB，支持CUDA）

- 硬盘：SSD≥128GB，空闲空间≥15GB（安装软件+模型存储+缓存+作品）

- 显卡驱动：有NVIDIA显卡则≥470.xx版本，无显卡则无要求

- 功能限制：单次批量制作≤10条文案，自定义数字人/音色模型单文件≤50MB，禁止多任务并行

- 性能表现：单条10-60秒视频生成耗时1-3分钟，纯CPU推理，口型同步、视频合成速度较慢

# 四、产品整体架构

## 4.1 界面布局

采用「左侧固定功能菜单+右侧核心内容区」Windows桌面端经典布局，左侧为8大核心模块入口，右侧为对应模块的操作界面，整体风格简洁明了、无冗余，符合Windows用户操作习惯，支持浅色/深色主题一键切换，切换后实时生效。

## 4.2 核心模块划分（8大模块，独立联动，围绕视频制作核心场景）

1. 首页模块：软件唯一入口，提供核心功能快速跳转、最近操作记录、新手教程引导，实时展示硬件状态与GPU加速情况。

2. 单条制作模块：核心功能模块，采用5步步骤化引导+右侧实时预览布局，实现单条数字人口播视频全流程制作。

3. 批量制作模块：支持文案批量导入、统一配置参数，本地批量生成多条口播视频，适配批量创作需求，按硬件等级限制单次生成条数。

4. 数字人管理模块：管理官方数字人与用户自定义数字人，支持预览、收藏、使用、上传、适配，确保口型同步效果。

5. 音色管理模块：管理内置CosyVoice3-0.5B衍生音色与用户自定义上传音色，支持试听、收藏、使用、上传、适配、删除、重新加载，实现模型本地加载与管理，释放硬盘空间，满足用户个性化音色需求。

6. 作品库模块：本地统一管理所有生成的口播视频，支持查看、播放、编辑、删除、批量操作，重新编辑可精准加载原始配置。

7. 设置模块：配置软件运行参数、修改模型/视频存储路径、优化性能设置、查看硬件信息、清理缓存、重置所有设置。

8. 帮助与反馈模块：提供本地离线使用教程、常见问题（FAQ）解答、客服入口、问题反馈通道，解决用户使用过程中的各类问题。

## 4.3 核心技术流程（本地全离线，无云端依赖，全流程本地完成）

1. 文案处理：用户输入/导入文案 → 本地文案优化引擎（自动断句、口语化优化、字数校验，耗时≤3秒）；

2. 语音合成：用户选择音色（内置/自定义）与参数 → 内置CosyVoice3-0.5B模型或自定义音色模型本地推理（生成带情感的WAV语音文件，CPU/GPU自适应加载）；

3. 口型同步：用户选择数字人 → 本地Wav2Lip口型同步引擎（将语音与数字人视频匹配，口型同步误差≤0.1秒，NVIDIA GPU加速）；

4. 视频合成：用户配置背景、字幕、BGM等参数 → 本地FFmpeg视频合成引擎（整合所有素材，NVIDIA NVENC硬件编码，生成1080P MP4视频）；

5. 作品管理：生成的视频自动保存至本地指定路径，同步至作品库，支持后续编辑、播放、导出、删除等操作。

4. 基础视频参数：背景设置下方，包含两项核心参数：

9. 字幕样式自定义（开启后可配置，实时预览）：

13. 字幕位置：支持“底部居中、底部左对齐、底部右对齐”，默认底部居中，点击位置选项实时更新预览效果。

14. 背景音乐（BGM）设置（默认关闭，避免盖过人声）：字幕设置下方，包含开关、内置BGM、自定义BGM和音量调节：

15. BGM开关：一键开启/关闭，开启后加载内置BGM列表，关闭后停止播放所有BGM；

16. 内置BGM：10首免费可商用BGM，按“轻快、舒缓、大气”分类，点击BGM名称即可选择，右侧预览区同步播放，支持暂停、重新播放；

17. 自定义BGM：点击“上传BGM”按钮，支持上传本地音频（MP3/WAV格式），上传后可预览、删除，上传文件最大≤100MB；

18. 音量独立调节：分别设置“人声音量”和“BGM音量”滑动条（调节范围0-200%），默认人声音量100%、BGM音量50%，调节时实时更新预览效果，确保人声（内置/自定义音色）清晰、BGM不盖过人声。

19. 操作按钮：左侧设置栏下方设置两个按钮，分别为：

20. 上一步：点击返回步骤3（数字人选择），保留当前所有视频设置（背景、比例、字幕、BGM），无需重新配置；

21. 生成视频：点击后跳转至步骤5（生成视频），开始执行视频生成流程，保留当前所有配置。

- 右侧预览区为核心展示区域，所有配置修改（文案、音色、数字人、视频参数）1秒内同步更新，无延迟、无模糊；自定义音色选择后，预览语音同步更新，与内置音色预览体验一致；

- 所有按钮状态清晰，不可点击时置灰，Hover时0.2秒内显示不可点击原因（如“请选择音色后再下一步”“剪贴板无可用文本”）；

- 所有弹窗（确认弹窗、提示弹窗）可拖动，关闭后记忆状态，再次触发时可恢复至上次关闭位置。

## 5.3 批量制作模块

### 5.3.1 功能描述

适配MCN、企业、矩阵号工作室的批量创作需求，支持批量导入多篇文案，统一配置音色（内置/自定义）、数字人、视频参数，本地批量生成多条口播视频，根据硬件配置等级动态限制单次生成条数，生成失败自动跳过并汇总结果，确保批量创作高效、便捷，无云端依赖。

### 5.3.2 具体功能

1. 文案批量导入（支持两种方式，导入耗时≤10秒，无卡顿，导入后自动校验）：

2. 手动输入/粘贴：左侧文案输入区，支持一行一个文案（换行分隔），批量粘贴多条文案，粘贴后自动拆分每行文案，标注序号；

3. 本地TXT导入：点击“导入TXT”按钮，支持导入本地TXT文件（要求一行一个文案，无多余格式），单次最大导入条数根据硬件等级动态限制：基础档≤10条、流畅档≤50条、高效档≤100条，超过限制弹出提示“请分批次导入，单次最多可导入X条文案”（X为对应硬件等级的最大条数）。

4. 文案批量管理与校验（导入后实时校验，无延迟）：

5. 列表展示：导入的所有文案按序号排列，每条文案标注字数、状态（有效/无效），无效文案标红显示，便于用户快速识别；

6. 自动校验规则：无效文案定义为“字数＜10字、空白文案、特殊字符（非中文/英文/数字/常用标点）占比超过50%”，校验完成后提示“有效文案X条，无效文案X条，请编辑/删除无效文案后生成”；

7. 文案操作：支持单条编辑（点击文案可修改）、单条删除（点击每条文案右侧“删除”图标）、批量勾选删除（勾选文案前的复选框，点击“批量删除”按钮）、一键清空（点击“清空所有”按钮），所有删除操作均弹出确认弹窗，防止误操作。

8. 统一参数配置（所有有效文案共用一套配置，与单条制作模块完全一致，降低操作成本，支持自定义音色配置）：

9. 配置内容：包含语音配置（音色选择：内置/自定义均可、语速/音量/情感强度调节）、数字人配置（官方/自定义数字人选择）、视频配置（背景、视频比例、字幕、BGM设置），配置界面与单条制作模块一致；

10. 配置预览：点击“预览配置”按钮，展示第一条有效文案的视频效果（文案+音色+数字人+视频参数），若选择自定义音色，预览时同步播放该音色的语音效果，用户可根据预览效果调整配置，调整后实时更新预览；

11. 自定义音色适配：选择自定义音色后，系统自动校验该音色模型完整性，若模型异常，弹出提示“所选自定义音色模型不完整，请重新上传或选择其他音色”，并强制切换至内置音色，确保批量生成正常执行。

12. 批量生成设置（本地持久化，用户配置后下次使用无需重新设置）：

13. 生成顺序：默认按文案导入序号依次生成，不可修改（确保文案与视频一一对应）；

14. 保存路径：默认与单条制作模块的视频保存路径一致，点击“选择路径”按钮可修改，支持新建文件夹（弹出Windows原生文件夹选择弹窗），修改后所有批量生成的视频均保存至新路径；

15. 视频命名：默认命名规则“视频+序号.mp4”（如视频1.mp4、视频2.mp4），支持自定义前缀（如“团购视频+序号.mp4”），前缀仅支持中文、英文、数字，输入特殊字符时提示“前缀仅支持中文、英文、数字，请修改”。

16. 批量生成流程（全程本地运行，无云端依赖，根据硬件等级自动适配推理模式）：

17. 进度展示：页面上方展示整体进度条（0-100%），进度条下方显示实时文本（“正在生成第X条/共Y条，当前步骤：XXX”），若选择自定义音色，语音合成步骤标注“自定义音色语音合成中”；下方列表展示每条文案的生成状态（进行中/成功/失败），失败条目标红显示，标注失败原因（如“模型加载失败”“硬件配置不足”“自定义音色模型异常”）；

18. 操作按钮：进度条下方设置两个按钮，分别为：

19. 暂停/继续：点击“暂停”，批量生成流程暂停，按钮变为“继续”，再次点击恢复生成，从中断的那条文案继续，无需重新生成已完成的视频；

20. 取消：点击“取消”，弹出确认弹窗“确定要取消批量生成吗？已生成的X条视频将保留，未生成的将终止”，点击“确认”后终止流程，未生成的文案不再执行，已生成的视频保存至指定路径。

21. 推理模式：根据硬件配置等级自动选择，高效档/流畅档开启“NVIDIA GPU批量并行加速”，基础档开启“CPU顺序推理”，确保生成效率最大化；选择自定义音色时，推理模式不变，仅语音合成环节调用自定义音色模型。

22. 生成结果汇总（批量生成完成后，2秒内弹出汇总提示）：

23. 生成成功：弹出提示“批量生成完成！共生成X条视频，已保存至指定路径，可前往作品库查看”，提示下方设置“前往作品库”按钮，点击直接跳转至作品库模块；

24. 生成失败：单条文案生成失败时
