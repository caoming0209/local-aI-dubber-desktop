# 二、产品整体架构

## 2.1 整体模块划分（8大核心模块）

产品采用「左侧固定菜单+右侧内容区」的布局，按功能划分为8大模块，模块间独立且可联动，全程围绕「视频制作」核心场景展开，具体模块如下：

1. 首页模块：快速入口、最近操作、教程引导

2. 单条制作模块：核心功能，实现单条数字人口播视频的全流程制作

3. 批量制作模块：实现多条视频的批量导入、统一配置、批量生成

4. 数字人管理模块：官方数字人展示、自定义数字人上传与管理

5. 音色管理模块：官方音色展示、试听、收藏与模型管理

6. 作品库模块：已生成视频的存储、查看、编辑与删除

7. 设置模块：软件参数配置、模型管理、缓存清理等

8. 帮助与反馈模块：使用教程、常见问题、客服入口

## 2.2 核心技术流程（本地离线推理）

全程不依赖云端，所有操作在本地完成，技术流程如下：

1. 用户输入/导入文案 → 本地文案优化引擎（自动断句、口语化优化）；

2. 选择音色与参数 → 本地语音合成模型（生成带情感的语音文件WAV）；

3. 选择数字人 → 本地口型同步引擎（Wav2Lip），将语音与数字人视频匹配，实现口型精准同步；

4. 配置视频参数（背景、字幕、BGM） → 本地视频合成引擎（FFmpeg），整合所有素材；

5. 生成并保存最终MP4视频 → 同步至作品库。

## 2.3 运行环境要求

### 2.3.1 最低配置（保证基础运行）

- CPU：Intel i5 或 AMD 同等性能及以上

- 内存：8GB RAM

- 硬盘：至少5GB空闲空间（用于安装软件及基础模型）

- 系统：Windows 10（64位）/ Windows 11（64位）

- 网络：首次使用需联网下载模型（后续可离线运行）

### 2.3.2 推荐配置（保证生成速度与视频质量）

- CPU：Intel i7 或 AMD 同等性能及以上

- 内存：16GB RAM 及以上

- 硬盘：10GB及以上空闲空间（SSD优先，提升模型加载与视频生成速度）

- 显卡：NVIDIA 独立显卡（显存≥4GB），支持CUDA加速（提升口型同步与视频合成速度）

- 系统：Windows 11（64位）

# 三、详细功能需求（按模块划分）

## 3.1 首页模块

### 3.1.1 功能描述

作为软件入口，提供快速操作指引、最近使用记录、核心功能入口，降低用户学习成本，提升操作效率。

### 3.1.2 具体功能

1. 欢迎语展示：根据用户使用时长，展示个性化欢迎语（如“欢迎回来，今天也要高效创作哦～”）；

2. 快速操作按钮（居中展示，大按钮强引导）：
        

    - 新建视频：点击直接跳转至「单条制作」模块；

    - 批量制作：点击直接跳转至「批量制作」模块；

    - 查看作品：点击直接跳转至「作品库」模块；

3. 最近使用记录：展示最近3条生成的视频/编辑的文案，点击可直接打开对应内容（视频打开播放，文案跳转至单条制作模块加载）；

4. 使用教程入口：右上角“教程”图标，点击弹出弹窗，展示3步快速制作教程（图文结合，可关闭、可重复查看）；

5. 软件版本信息：左下角展示当前软件版本号，鼠标hover显示更新状态（如“当前为最新版本”“有新版本可更新”）；

6. 帮助入口：左下角“帮助”图标，点击跳转至「帮助与反馈」模块。

### 3.1.3 交互要求

- 所有按钮点击有明显反馈（颜色变深、轻微缩放），无卡顿；

- 最近使用记录hover时显示操作选项（播放/编辑/删除），点击对应选项执行操作；

- 教程弹窗可拖动、可关闭，关闭后再次点击教程入口可重新打开；

- 左侧菜单切换时，首页内容区同步切换，切换无延迟、无闪烁。

## 3.2 单条制作模块（核心模块）

### 3.2.1 功能描述

实现单条数字人口播视频的全流程制作，从文案输入到视频生成、预览，流程清晰，操作极简，支持自定义配置，满足不同用户的视频需求。

### 3.2.2 界面布局

采用「步骤化引导+右侧实时预览」布局，左侧为5步制作流程（文案输入→语音选择→数字人选择→视频设置→生成视频），右侧为实时预览区（同步展示当前配置的效果）。

### 3.2.3 具体功能（按制作步骤）

#### 步骤1：文案输入

1. 文案输入框：支持手动输入、粘贴文案，文本框可拉伸，支持换行；

2. 字数提示：实时显示当前文案字数，提示“建议输入50-500字（适配10-60秒口播视频）”；

3. 文案操作按钮：
        

    - 清空：一键清空当前输入的文案（点击弹出确认弹窗，防止误操作）；

    - 复制：一键复制当前文案；

    - 粘贴：一键粘贴剪贴板中的文案（若剪贴板无文本，按钮置灰不可点击）；

4. 文案优化功能：
       

    - 自动断句：输入文案后，点击“自动断句”，根据口语化习惯拆分句子（如逗号、句号处拆分，适配口播节奏）；

    - 口语化优化：点击“口语化优化”，将书面化文案转换为口语化表达（如“因此”改为“所以”，“综上所述”改为“总结一下”），优化后可还原为原文案；

5. 文案模板库：
        

    - 模板分类：好物推荐、知识讲解、团购探店、企业宣传、剧情口播（5类核心模板）；

    - 模板使用：点击对应模板，文案输入框自动加载模板内容，用户可直接修改使用；

    - 模板预览：hover模板展示模板预览文案（部分内容，点击可查看完整内容）；

6. 下一步按钮：文案输入≥10字可点击“下一步”，跳转至步骤2（语音选择）；若文案＜10字，点击提示“请输入至少10字文案”。

#### 步骤2：语音选择

1. 音色分类展示：按“男声、女声、情感音、方言音”分类，每个分类下展示对应音色（带音色名称、简要描述，如“男声-沉稳：适合专业讲解”）；

2. 音色操作：
        

    - 试听：每个音色右侧有“试听”按钮，点击播放该音色的示范语音（默认播放“欢迎使用智影口播助手，本工具致力于为您提供智能、流畅、高质感的口播体验，助力您快速打造优质音频与视频内容，提升创作效率。”）；

    - 选择：点击音色，该音色高亮显示（标注“已选择”），同时右侧预览区同步播放当前文案的语音预览（仅播放前10秒，完整语音需生成后试听）；

    - 收藏：每个音色右侧有“收藏”图标，点击收藏，收藏后的音色显示在“我的收藏”分类下，方便快速选择；

3. 语音参数调节（滑动条调节，实时生效，预览区同步展示效果）：
        

    - 语速：调节范围0.5-2.0（默认1.0，0.5最慢，2.0最快）；

    - 音量：调节范围0-200%（默认100%，0为静音，200%为最大音量）；

    - 情感强度：调节范围0-100%（默认50%，仅情感音可调节，非情感音该选项置灰）；

4. 操作按钮：
        

    - 上一步：点击返回步骤1（文案输入），保留当前文案内容；

    - 下一步：选择音色后可点击“下一步”，跳转至步骤3（数字人选择）；若未选择音色，点击提示“请选择一款音色”；

#### 步骤3：数字人选择

1. 数字人分类展示：按“男主播、女主播、职场、亲和、专业”分类，每个分类下展示数字人预览图（高清，尺寸统一）；

2. 数字人操作：
       

    - 预览：hover数字人预览图，自动播放该数字人的动作小样（3秒循环，无语音）；点击预览图，右侧预览区同步展示该数字人的实时效果（结合当前文案的语音，口型同步预览，仅播放前10秒）；

    - 选择：点击数字人，该数字人高亮显示（标注“已选择”），同步更新右侧预览区；

    - 收藏：每个数字人右侧有“收藏”图标，点击收藏，收藏后的数字人显示在“我的收藏”分类下，方便快速选择；

3. 自定义数字人入口：右上角“上传自定义数字人”按钮，点击跳转至「数字人管理模块」的自定义上传页面（上传完成后，自动返回当前步骤，同步显示自定义数字人）；

4. 操作按钮：
        

    - 上一步：点击返回步骤2（语音选择），保留当前音色与参数配置；

    - 下一步：选择数字人后可点击“下一步”，跳转至步骤4（视频设置）；若未选择数字人，点击提示“请选择一位数字人”；

#### 步骤4：视频设置

1. 背景设置：
        

    - 背景分类：纯色、办公场景、室内场景、虚化背景（4类）；

    - 背景选择：点击对应背景，右侧预览区同步更新；

    - 自定义背景：支持上传本地图片作为背景（点击“上传背景”，选择本地图片，支持JPG、PNG格式，上传后自动适配视频比例）；

2. 视频参数设置：
       

    - 分辨率：默认1080P（不可修改，保证视频清晰度）；

    - 视频比例：支持切换“竖屏9:16（抖音/视频号）”“横屏16:9（B站/西瓜视频）”，切换后右侧预览区同步调整比例；

3. 字幕设置：
        

    - 字幕开关：默认开启，关闭后视频无字幕；

    - 字幕样式配置（开启字幕后可配置）：
    

        - 字体：支持选择系统自带字体（如微软雅黑、宋体、黑体等，默认微软雅黑）；

        - 字体大小：调节范围20-40号（默认30号）；

        - 字体颜色：支持选择预设颜色（黑、白、红、蓝等），也可自定义颜色（取色器）；

        - 字幕位置：支持“底部居中、底部左对齐、底部右对齐”（默认底部居中）；

    - 字幕预览：所有配置实时生效，右侧预览区同步展示字幕效果；

4. 背景音乐设置：
        

    - BGM开关：默认关闭，开启后加载内置BGM；

    - 内置BGM：提供10首免费可商用BGM（分类：轻快、舒缓、大气），点击选择，右侧预览区同步播放；

    - 自定义BGM：支持上传本地音频作为BGM（点击“上传BGM”，选择本地音频，支持MP3、WAV格式）；

    - 音量调节：分别调节“人声音量”“BGM音量”（滑动条，0-200%，默认人声100%、BGM50%），避免BGM盖过人声；

5. 操作按钮：
        

    - 上一步：点击返回步骤3（数字人选择），保留当前数字人选择；

    - 生成视频：点击“生成视频”，跳转至步骤5（生成视频），开始执行视频生成流程；

#### 步骤5：生成视频

1. 生成进度展示：
        

    - 进度条：显示整体生成进度（0-100%）；

    - 进度文本：实时显示当前生成步骤与进度（如“文案优化中...20%”“语音合成中...40%”“口型同步中...60%”“视频合成中...80%”“生成完成100%”）；

2. 操作按钮：
       

    - 暂停：生成过程中可点击“暂停”，暂停后按钮变为“继续”，点击继续生成；

    - 取消：生成过程中可点击“取消”，点击后弹出确认弹窗（“确定要取消生成吗？已生成内容将不保存”），确认后终止生成流程，返回步骤4（视频设置）；

3. 生成完成后操作：
        

    - 提示信息：生成完成后，弹出提示“视频生成成功！已保存至作品库”；

    - 播放：点击“播放”，右侧预览区播放生成的完整视频；

    - 打开文件位置：点击“打开文件位置”，自动弹出视频保存的本地文件夹；

    - 重新生成：点击“重新生成”，返回步骤4（视频设置），可修改配置后重新生成；

    - 制作下一条：点击“制作下一条”，返回步骤1（文案输入），清空当前文案，开始新视频制作；

4. 生成失败处理：

    - 提示信息：生成失败后，弹出提示“视频生成失败，请检查以下内容：1. 模型是否下载完整；2. 硬件配置是否达标；3. 文案是否合规”；

    - 重试按钮：点击“重试”，重新执行生成流程；

    - 帮助按钮：点击“帮助”，跳转至常见问题页面（查看生成失败的解决方案）；

### 3.2.4 交互要求

- 步骤跳转流畅，无卡顿、无闪烁，保留上一步配置；

- 右侧预览区实时同步当前配置（文案修改、音色切换、数字人选择、视频设置调整后，预览区1秒内更新）；

- 所有按钮点击有反馈，不可点击状态下置灰，鼠标hover显示不可点击原因；

- 生成视频过程中，进度条与进度文本实时更新，无卡顿；暂停/取消操作响应及时（1秒内）；

- 文案优化、语音合成、口型同步等操作，若耗时超过3秒，显示加载动画（避免用户误以为软件卡死）。

## 3.3 批量制作模块

### 3.3.1 功能描述

支持用户批量导入多篇文案，统一配置语音、数字人、视频参数，批量生成多条视频，提升矩阵号、批量宣传视频的制作效率。

### 3.3.2 具体功能

1. 文案批量导入：

    - 导入方式1：手动输入（一行一个文案，换行分隔，支持粘贴多行文案）；

    - 导入方式2：导入本地TXT文件（TXT文件中一行一个文案，支持最大100条文案/文件）；

    - 文案管理：
                

        - 查看：导入后展示所有文案（每条文案标注序号）；

        - 编辑：点击单条文案可修改内容；

        - 删除：点击单条文案右侧“删除”按钮，删除该文案（支持批量勾选删除）；

        - 清空：一键清空所有导入的文案（点击弹出确认弹窗）；

    - 文案校验：导入后自动校验，提示“有效文案X条，无效文案X条”（无效文案：＜10字、空白文案），无效文案标注红色，可单独删除或编辑；

2. 统一配置（所有有效文案共用一套配置，无需单独设置）：
        

    - 语音配置：选择音色、调节语速、音量、情感强度（与单条制作模块一致）；

    - 数字人配置：选择数字人（与单条制作模块一致）；

    - 视频配置：选择背景、视频比例、字幕设置、BGM设置（与单条制作模块一致）；

    - 预览：点击“预览”，展示第一条文案的视频效果（与单条制作的预览一致），可根据预览调整配置；

3. 批量生成设置：
        

    - 生成顺序：按文案导入顺序生成（标注序号）；

    - 保存路径：可自定义视频保存路径（默认与单条制作一致，点击“选择路径”修改）；

    - 视频命名：默认“视频+序号.mp4”（如视频1.mp4、视频2.mp4），支持自定义前缀（如“团购视频+序号.mp4”）；

4. 批量生成流程：
        

    - 进度展示：显示整体生成进度（0-100%）、当前生成条数（如“正在生成第3条/共10条”）、当前生成步骤；
